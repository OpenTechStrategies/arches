Arches 8.0.5 Release Notes
--------------------------

### Bug Fixes and Enhancements
- Fix bug related case sensitivity in `GraphXPublishedGraph.find_publication_in_language()` [#12457](https://github.com/archesproject/arches/pull/12457)
- Hide "Skip to print" link in the printed version of page [#12467](https://github.com/archesproject/arches/pull/12467)

### Dependency changes:
```
Python:
    Upgraded:
        None

JavaScript:
    Upgraded:
        none
```

### Upgrading Arches

1. Upgrade to version 8.0.0 before proceeding by following the upgrade process in the [Version 8.0.0 release notes](https://github.com/archesproject/arches/blob/dev/8.0.x/releases/8.0.0.md)

2. Upgrade to Arches 8.0.5
    ```
    pip install --upgrade arches==8.0.5
    ```

3. If you are running Arches on Apache, restart your server:
    ```
    sudo service apache2 reload
    ```
