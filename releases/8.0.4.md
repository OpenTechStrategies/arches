Arches 8.0.4 Release Notes
--------------------------

### Bug Fixes and Enhancements
- Fix bug related to scrolling for bulk importers [#12398] (https://github.com/archesproject/arches/pull/12398)
- Fix resource editor failure to render [#12408](https://github.com/archesproject/arches/pull/12408)
- Fix issue with the attached files not found for the branch/tile excel importers [#12426](https://github.com/archesproject/arches/pull/12426)
- Fix issue where FORCE_SCRIPT_NAME not being prepended to generated URLs [#12434](https://github.com/archesproject/arches/pull/12434)
- Add workaround for PrimeVue theme import issues after upgrading to v1.20+ [#12443](https://github.com/archesproject/arches/issues/12443)

### Upgrading Arches

1. Upgrade to version 8.0.0 before proceeding by following the upgrade process in the [Version 8.0.0 release notes](https://github.com/archesproject/arches/blob/dev/8.0.x/releases/8.0.0.md)

2. Upgrade to Arches 8.0.4
    ```
    pip install --upgrade arches==8.0.4
    ```

3. If you are running Arches on Apache, restart your server:
    ```
    sudo service apache2 reload
    ```
